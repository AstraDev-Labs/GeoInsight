# GeoForesight — Roadmap

> A living document outlining planned features and improvements for the GeoForesight Intelligence Platform.

---

## 🟢 Completed

- [x] Core blog publishing pipeline (request → review → publish)
- [x] DynamoDB + local JSON hybrid data layer
- [x] Cloudflare R2 file storage with public URL support
- [x] Rich text editor with image/attachment uploads
- [x] Admin dashboard with analytics, request management, and content review
- [x] Collaborative comment system with user registration
- [x] Bot auto-moderation (violation terms, muting, banning)
- [x] SEO: structured data, Open Graph, sitemaps, IndexNow
- [x] Sentry error monitoring & Axiom web vitals
- [x] Automatic R2 file cleanup on post decline/delete
- [x] Website lockdown modes (Maintenance & Technical Difficulties)

---

## 🔵 In Progress / Next Up

- [ ] **Custom lockdown messages** — Allow admins to write custom messages for lockdown screens
- [ ] **Scheduled maintenance windows** — Set a future time for automatic lockdown activation
- [ ] **Author notifications** — Email authors when their post is published or declined
- [ ] **Post editing** — Allow authors to submit edits to published posts

---

## 🟡 Planned

- [ ] **User dashboard** — Let authors track their submission status
- [ ] **Category management** — Admin CRUD for blog categories
- [ ] **Image gallery** — Browse and manage all uploaded assets from admin
- [ ] **Search functionality** — Full-text search across published posts
- [ ] **Comment email notifications** — Notify users of replies to their comments
- [ ] **Dark mode** — Site-wide dark theme toggle
- [ ] **RSS feed** — Auto-generated RSS for blog subscribers
- [ ] **API rate limiting** — Protect endpoints from abuse

---

## 🔴 Future / Exploratory

- [ ] **Multi-author collaboration** — Multiple reviewers and editors per post
- [ ] **GIS map integration** — Interactive maps embedded in blog posts
- [ ] **AI-assisted content review** — Automated quality checks on submissions
- [ ] **Internationalization (i18n)** — Multi-language support
- [ ] **PWA support** — Offline reading and push notifications
- [ ] **Webhook integrations** — Slack/Discord notifications on new submissions
